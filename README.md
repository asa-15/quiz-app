# quiz-app
A four-choice quiz game using anime and game music, created as a school project.

# ゲーム・アニメ音楽クイズ

## 概要

1年生の進級作品として制作した、アニメやゲームの音楽を使ったクイズゲームです。  
クイズは4択形式で、流れる音楽を聴いてその作品名やゲームタイトルを当てるスタイルです。
銅賞と最多得票賞を受賞し、多くの方に評価されました。

## 特徴

- 「楽しんでもらえるものを作りたい」という思いから、アニメやゲーム音楽をテーマにしたクイズを企画。  
- 実際に遊んでもらうことを大切にし、積極的に声をかけて多くの人にプレイしてもらえるよう工夫しました。  
- 最終的に銅賞と最多得票賞を受賞し、多くの人に楽しんでもらうことができました。

## ゲーム内容

- クイズは4択形式で、音楽を聴いて正解を選ぶスタイルです。  
- 最後に、今までのプレイヤーの正答率を横棒グラフで表示し、正解率・不正解率を視覚的にわかりやすくしました。  

## 使用技術

- HTML / CSS / JavaScript

## 工夫した点・技術的なポイント
 
- 授業で配布されたコードを基に、自分で改良を重ね、JavaScriptの挙動を独自にカスタマイズ。  
- ユーザーが結果を直感的に理解できるよう、グラフの動きの調整やUIにこだわりました。
- どんなに良い作品でも多くの人に遊んでもらわなければ意味がないと考え、積極的に声かけを行い、プレイしてもらう機会を増やしました。
- この経験を通じて、技術力だけでなく企画力やコミュニケーション力も磨きました。

- ※本リポジトリには著作権保護の観点から、音楽ファイルおよび画像ファイル（imagesフォルダ）は含まれておりません。
